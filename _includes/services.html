How it works.. ?

Pointer - Movements

Keep track of the movement of the mouse or touchpad

Functioning

Detect who accesses your device(laptop or server other iot device) with mouse or touchpad installed

Sends Warning

Send warning messages on Twitter/Twilio-SMS/Slack/Telegram. More features on the way.

Installation

PyPi

You can install SecureTea from PyPi package manager

pip install securetea

Please make sure all dependencies are installed if this fails.

Github

Installing from GitHub involves the following steps:

git clone https://github.com/OWASP/SecureTea-Project.git

Install SecureTea package

sudo python setup.py install

Install python dependencies

pip install -r requirements.txt

Zip

Download the Zip

Unzip using: unzip master.zip

Navigate into the project directory

cd SecureTea-Project

Install SecureTea package

sudo python setup.py install
